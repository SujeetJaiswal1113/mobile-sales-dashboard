# 📊 Mobile Sales Performance Dashboard (Power BI)

## 🚀 Project Overview
This project analyzes mobile sales data using Power BI to generate business insights.

---

## 🔍 Key Insights
- Total Sales reached 769M with peak performance in July
- Sales increased from February to mid-year
- Apple leads in total sales and transactions
- Xiaomi shows consistent monthly sales
- UPI and Debit Card are most preferred payment methods
- Majority ratings are 4 and 5
